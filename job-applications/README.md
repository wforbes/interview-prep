# Job Applications Tracker

Track your job applications and interview progress here.

## Organization

- **applied/** - Applications that have been submitted
- **interviewing/** - Active interview processes
- **archived/** - Completed processes (offers, rejections, withdrawn)

## Template

Use this template for each application (filename: `{company}-{role}.md`):

```markdown
# {Company Name} - {Role Title}

**Status:** Applied | Phone Screen | Technical Interview | Onsite | Offer | Rejected | Withdrawn
**Applied Date:** YYYY-MM-DD
**Job Posting URL:** [Link]
**Salary Range:** $XXX,XXX - $XXX,XXX

## Role Details

### Requirements
- Requirement 1
- Requirement 2
- Requirement 3

### Nice to Have
- Item 1
- Item 2

### Tech Stack
- Technology 1
- Technology 2

## Why This Role?

[Your notes on why you're interested in this position]

## Interview Timeline

| Date | Stage | Notes |
|------|-------|-------|
| YYYY-MM-DD | Applied | Submitted via company website |
| YYYY-MM-DD | Phone Screen | Talked with recruiter, discussed... |
| YYYY-MM-DD | Technical Round 1 | LeetCode medium problem, system design... |

## Preparation Notes

### Topics to Review
- [ ] Topic 1
- [ ] Topic 2

### Questions to Ask
- Question 1?
- Question 2?

## Follow-up Actions

- [ ] Send thank you email
- [ ] Review feedback
- [ ] Next steps

## Outcome

[Final result and any notes for future reference]
```

## Tips

1. Move files between folders as status changes
2. Keep notes detailed for future reference
3. Track patterns in interview questions
4. Note what went well and what to improve
5. Record salary discussions and negotiations
